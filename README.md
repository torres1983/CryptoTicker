# 📈 CryptoTicker for macOS

一个轻量级、纯原生的 macOS 状态栏应用程序，提供无缝的加密货币实时盯盘体验。直连币安 (Binance) API，让你在处理其他工作或进行复杂图表分析时，核心关注标的的价格异动一目了然。

对于需要时刻紧盯盘面、捕捉微小波动的短线交易者而言，这是一款完美的“外挂”小工具。

## ✨ 核心特性 (Features)

* **⚡️ 实时永续合约数据**: 直连币安 U 本位合约 API (`fapi.binance.com`)，获取最敏锐的市场价格。
* **📌 一键置顶 (Pin to Menu Bar)**: 下拉菜单中点击任意币种，瞬间将其“钉”在系统状态栏主位。
* **🚦 涨跌视觉反馈**: 状态栏文本根据价格波动实时变色（🟢 上涨 / 🔴 下跌），扫一眼即可感知多空力量。
* **⚙️ 极简配置面板**: 原生 macOS 设置窗口，支持随时添加或删除关注的交易对，即刻生效，无需重启。
* **🪶 极致轻量**: 采用 Swift + AppKit + SwiftUI 原生混合开发，零第三方依赖，极低 CPU 与内存占用。

## 🚀 快速开始 (Getting Started)

### 环境要求
* macOS 13.0+
* Xcode 15.0+ (用于编译)
* Swift 5+

### 编译与运行
1.  克隆本仓库到本地计算机。
2.  使用 Xcode 打开 `CryptoTicker.xcodeproj`。
3.  在 Xcode 顶部选择你的 Mac 作为运行目标 (`Any Mac` 或 `My Mac`)。
4.  按下 `Cmd + R` 编译并运行。
5.  *(可选)* 点击 Xcode 菜单栏的 **Product -> Archive**，将其导出为独立的 `.app` 文件，以便放入“应用程序”文件夹并设置开机自启。

## 🛠️ 配置与使用 (Usage)

应用启动后，会作为一个后台服务静默运行在右上角状态栏。

1.  **设置关注列表**: 点击状态栏图标，选择下拉菜单中的 `设置 (Preferences)...`。
2.  **输入交易对**: 在文本框中输入你关注的币安合约交易对，以英文逗号分隔。
    * *示例格式*: `SOLUSDT,BTCUSDT,ETHUSDT`
3.  **切换主视角**: 在下拉菜单中直接点击某个币种（例如 ETH），即可将其替换为状态栏的常驻显示标的，该标的前方会出现 `★` 标识。

> **注意**: 请务必使用带有 `USDT` 后缀的标准币安交易对格式（大小写均可，系统会自动转换）。

## 🏗️ 技术栈 (Tech Stack)

* **Swift Concurrency (`async/await`)**: 处理非阻塞的底层网络请求。
* **AppKit (`NSStatusItem`, `NSMenu`)**: 构建纯净、系统级的下拉菜单交互。
* **SwiftUI (`AppStorage`, `Settings`)**: 构建现代化的响应式设置面板与数据持久化。
* **Binance Futures API**: 提供稳定且无频次限制的公开行情数据。
